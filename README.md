# Car Price Predictor

A Machine Learning web application that predicts the price of used cars based on features like company, model, year, fuel type and kilometers driven.

##  Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Bootstrap

##  Features
- Predict used car price using ML model
- Simple web interface
- Real-time prediction

##  Project Structure

car-price-predictor
│
├── app.py
├── Cleaned_Car_data.csv
├── LinearRegressionModel.pkl
├── templates
│ └── index.html

##  Run Locally

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open browser:

http://127.0.0.1:5000
